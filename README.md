A responsive React app for uploading, editing, and managing photos with dynamic pricing. Users can crop, rotate, and revert edits on multiple images. The app calculates total price based on selected size, paper type, material, and number of images.

Features

Upload multiple images with previews

Crop and rotate images

Undo image edits

Select photo size, paper type, and material

Dynamic price calculation

Responsive layout with sticky bottom bar showing price details

Tech Stack

React.js, Tailwind CSS

react-easy-crop for image editing

React Router for navigation

Usage

Upload images using the upload area

Click an image to crop or rotate

Select size, paper type, and material

View total price at the bottom

Click NEXT to proceed
